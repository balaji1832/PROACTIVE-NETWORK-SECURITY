Guide to Analyzing Packets in Wireshark

1. Open Wireshark

Launch Wireshark on your system and ensure you have administrative privileges to capture all traffic.

2. Select a Network Interface

Choose the network interface you want to monitor from the list provided (e.g., Wi-Fi, Ethernet).

3. Start Capturing Packets

Click the blue shark fin icon at the top-left corner to start capturing packets. You will see packets appearing in real-time.

4. Analyze the Packet Details

Select a packet from the list to view its details. Look for the following information:

  • Source Address (src): Found under the IP layer; identifies the sender of the packet.
  • Destination Address (dst): Found under the IP layer; identifies the receiver of the packet.
  • Protocol Type: Listed in the Protocol column or in the IP layer (e.g., TCP, UDP, ICMP).
  • Flags: Found in the TCP or UDP layer, such as SYN, ACK, FIN, etc.
  • Service: Determined by the port numbers in the TCP/UDP layer (e.g., HTTP: Port 80, HTTPS: Port 443).

    • 5. Stop and Save

    Click the red square icon to stop the capture. Save the file if needed by navigating to File > Save As.
